# Set Up Entity Framework Core with SQLite

Configure Entity Framework Core with SQLite for development. This establishes the data layer foundation for the player service.

## Sub-tasks:

- [x] **Add EF Core packages**: Add `Microsoft.EntityFrameworkCore.Sqlite` and `Microsoft.EntityFrameworkCore.Design` to `player-service.csproj`
- [x] **Create AppDbContext**: Create `Data/AppDbContext.cs` with DbSet properties for Player, Item, and Inventory
- [x] **Configure connection string**: Add SQLite connection string to `appsettings.Development.json`
- [x] **Register DbContext**: Register `AppDbContext` in `Program.cs` with SQLite provider
- [x] **Verify setup**: Ensure the project builds successfully with EF Core configured
