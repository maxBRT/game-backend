# Project Setup

Initialize the benchmark-cli Go project with the required directory structure, dependencies, and basic scaffolding.

## Sub-tasks:

- [x] **Create project directory**: Create `/benchmark-cli` directory at the repository root
- [x] **Initialize Go module**: Run `go mod init benchmark-cli` to create go.mod
- [x] **Create directory structure**: Create `cmd/`, `internal/client/`, `internal/benchmark/`, `internal/dashboard/`, `internal/config/` directories
- [x] **Add cobra dependency**: Run `go get github.com/spf13/cobra@latest`
- [x] **Add bubbletea dependency**: Run `go get github.com/charmbracelet/bubbletea@latest`
- [x] **Add lipgloss dependency**: Run `go get github.com/charmbracelet/lipgloss@latest` (for TUI styling)
- [x] **Create main.go**: Create entry point that calls `cmd.Execute()`
- [x] **Verify build**: Run `go build` to ensure project compiles
